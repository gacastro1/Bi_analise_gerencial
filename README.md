# 📊 Projeto de BI - Análise de Informações Gerenciais

## 🎯 Objetivo
Este projeto tem como objetivo transformar dados brutos em informações estratégicas para suporte à tomada de decisão gerencial.

A proposta simula um cenário real de negócio, onde gestores precisam acompanhar indicadores-chave para entender desempenho, identificar problemas e direcionar ações.

## 📁 Fonte de Dados
Os dados utilizados neste projeto são:
- Saúde Populacional
- Dados simulados

---

## 🛠️ Ferramentas Utilizadas
- SQL (extração e tratamento de dados)
- Power BI (modelagem e visualização)

---

## 🔄 Processo de Desenvolvimento

### 1. Extração e Tratamento
- Criação de queries SQL para filtragem e organização dos dados
- Padronização e limpeza das informações

### 2. Modelagem de Dados
- Estruturação de tabelas relacionais
- Definição de relacionamentos entre entidades

### 3. Construção do Dashboard
- Desenvolvimento de visualizações interativas
- Criação de indicadores estratégicos

---

## 🧠 Insights Gerados
- A base é mais jovem do que o esperado para um plano corporativo
- Um critério simples revelou um grupo que merece atenção
- Quarta e quinta concentram quase metade de todos os atendimentos
  

## 📌 Considerações Finais
Este projeto faz parte do meu desenvolvimento na área de dados, com foco em Business Intelligence e análise estratégica.

Busco aplicar esse conhecimento principalmente em contextos reais, como saúde e operações, onde dados têm impacto direto na tomada de decisão.
